# Fruitop
A mordern responsive homepage about the fictious Fruitop, a ice cream company!

## About the project
The project is to practice modern/clean development of semantic HTML, clean CSS and JavaScript.
And to practice organization, creating issues, pull requests and resolving merge conflicts.
